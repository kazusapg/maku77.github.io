---
title: カレントディレクトリのパスを取得する
created: 2016-08-17
---

コマンドプロンプト上で `%CD%` を参照すると、カレントディレクトリのパスを取得することができます。

```
C:\foo\bar> echo %CD%
C:\foo\bar
```

`clip` コマンドを組み合わせて使用すると、カレントディレクトリのパスをクリップボードに流し込むことができます。

```
C:\foo\bar> echo %cd% | clip
```

これで、カレントディレクトリのパスを `Ctrl-C` で任意のアプリケーションに張り付けることができます。
