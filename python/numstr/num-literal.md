---
title: 2進数、8進数、16進数の数値リテラル
created: 2009-11-18
---

数値のプレフィックスとして、`0b`, `0o`, `0x` を付けることにより、十進表記以外で数値を表すことができます。

#### 2 進数

```python
>>> 0b0111
7
```

#### 8 進数

```python
>>> 0o777
511
```

#### 16 進数

```python
>>> 0xFFFF
65535
```

Python 3.0 以前は 8 進数を以下のように表現していましたが、Python 3.0 以降は SyntaxError になるので、`0o` を使用する必要があります。

#### 8 進数（旧式）

```python
>>> 0777
511
```
