---
title: 多次元配列を生成する
date: "2012-01-30"
---

配列の中に配列を含めることで、多次元配列を表現することができます。

~~~ php
$arr = array(
    array(1, 2, 3),
    array(4, 5, 6),
    array(7, 8, 9));
~~~

上記のようにすると、次のような要素を持つ多次元配列が生成されます。

~~~
[[1, 2, 3], [4, 5, 6], [7, 8, 9]]
~~~

以下のように、順番に要素を格納していっても同じ多次元配列を作成できます。

~~~ php
$arr[0][] = 1;
$arr[0][] = 2;
$arr[0][] = 3;
$arr[1][] = 4;
$arr[1][] = 5;
$arr[1][] = 6;
$arr[2][] = 7;
$arr[2][] = 8;
$arr[2][] = 9;
~~~
